Log In (auto-fetches refresh token)
Credentials are sent through your Cloudflare Pages proxy to fanstack.io and never stored — only the resulting refresh token is saved in this browser's localStorage. Google-login users can't use this; use the manual refresh-token paste below.
Refresh Token (Auto-refreshing, ~14 day lifespan)
How to get the refresh token
1. Log into fanstack.io.
2. Open DevTools (F12) → Application tab (Chrome) or Storage tab (Firefox).
3. Under Cookies → https://fanstack.io, find fanstack.refresh_token.
4. Copy the Value and paste here.

Caveat: each refresh rotates this token. If you're actively using fanstack.io in another tab and the app auto-refreshes, your fanstack.io tab may get logged out (you'll just need to re-login there and grab a fresh refresh token if you want auto-refresh to keep working).
Manual JWT (fallback)
How to get this token
Easy way (bookmarklet):
1. Drag this link to your bookmarks bar: 📋 Grab FanStack JWT
2. Open fanstack.io (must be logged in) and click the bookmark.
3. Come back here, paste, click Save Token.

Manual way (if bookmarklet fails):
1. Log into fanstack.io. Open DevTools (F12) → Network tab → filter link-pages.
2. Reload. Click any request → Headers → find authorization: Bearer eyJ....
3. Copy everything after Bearer and paste here.

The token is stored in this browser only; it's forwarded to fanstack.io through our proxy per request.
